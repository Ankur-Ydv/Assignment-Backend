## Installation Guide
### clone the project
```
  git clone git@github.com:Ankur-Ydv/Assignment-Frontend.git
```
### go to the project directory
```
  cd Assignment-Frontend
```
### Install dependencies
```
  npm install
  npm install react react-dom next
```
### start the app
```
  npm run dev
```
